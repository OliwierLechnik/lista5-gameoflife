◆ Cell()

org.example.gameoflife.Cell.Cell ( int ID)
inline

Constructs a new Cell instance with a unique ID, initial state, and random color. The constructor initializes the cell with the following properties:

  • The unique identifier of the cell is set to the provided ID.
  • The cell's state is set to active (not frozen).
  • The loop thread is initialized as null.
  • An empty list of neighboring cells is created.
  • The cell's color is set to a random RGB value.
Parameters
IDThe unique identifier for the new cell.

synchronized Color org.example.gameoflife.Cell.readWriteColor ( Color newColor)
inline

Performs a read and/or write operation on the cell's color. If a new color is provided, it updates the cell's color with the new value. Regardless of the update, it returns the previous color of the cell.

Parameters

Returns
The previous color of the cell before any update.

◆ switchState()

void org.example.gameoflife.Cell.switchState ( )
inline

Toggles the frozen state of the cell and prints the state change to the console. If the cell becomes frozen, it prints "Stop: [ID]". If the cell becomes active, it prints "Start: [ID]".
